Python Libraries

Open-source Python packages published on PyPI for geospatial analysis and remote sensing.

Desktop Apps

Note+

A lightweight cross-platform text editor built with Electron and Monaco Editor. Features: syntax highlighting for 40+ languages, tab management, find/replace, dark/light themes, session persistence.

Published Packages

sen2p

Search and download Sentinel-2 imagery from Microsoft Planetary Computer. Features: search by location/date/cloud cover, best item selection, band aliases, parallel download, CLI.
rasteric

Python geospatial library for raster preprocessing, analysis, visualization. ~30 functions: vegetation indices, mosaic, classification, time-series, change detection, cloud masking, ML prep.
